Excercise no 2
Write a function, evenOrOdd, that determines whether its argument is an even number. 
If it is, the function should log 'even' to the console; otherwise, it should log 'odd'. For now, assume that the argument is always an integer.

function evenOrOdd(numInput) {
  if (numInput % 2 === 0) {
    return console.log('even');
  } else {
    return console.log('odd');
  }
}
